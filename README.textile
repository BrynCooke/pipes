Pipes is a Java "dataflow":http://en.wikipedia.org/wiki/Dataflow framework in Java that supports the splitting, merging, filtering, and transformation of data from input to output in a "lazy":http://en.wikipedia.org/wiki/Lazy_evaluation, memory-efficient, "combinator":http://en.wikipedia.org/wiki/Function_composition fashion. Pipes provides specific support for traversing "Blueprints":http://blueprints.tinkerpop.com enabled "graph databases":http://en.wikipedia.org/wiki/Graph_database. As such, it can be used as a foundation for graph-based query/traversal langauges.

The documentation for Pipes can be found at this "location":http://pipes.tinkerpop.com. Finally, please visit "TinkerPop":http://tinkerpop.com for other software products.
